TOOLKIT FOR THE R^N GAME
{ FOR WHICH PEOPLE SEEM TO BE QUITE CRAZY RIGHT NOW }
Created by Abhishek Ghosh
Use ARROW KEYS to to move the tiles. When two tiles
with the same number touch, they merge into one!

Click on menu icon at the top right of the
window to open the Game Generator.

www.github.com/abhishekcghosh/RNGame

Based on the works of Gabriele Cirulli, which was inspired by 1024 by Veewo Studio
and conceptually similar to Threes by Asher Vollmer.

This code is developed from scratch but is identical in concept and created with an additional
agenda of developing an extensible framework for experimenting with the game itself.
